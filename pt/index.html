Questões

Todos nós temos as respostas.

Se é novo nos pagamentos em crypto, estas são as coisas que provavelmente vai querer saber.

Lê a documentação
Preciso de manter encriptação para usar isso? +
Não. A maioria dos comerciantes define a preferência de pagamento para conversão automática em euros. O cliente paga em crypto, nós fazemos a conversão, recebes euros. Nunca tocas numa carteira se não quiseres.
Quanto custa? +
Para Commerce e Invoice, taxa fixa de 1,99% por transação, sem mensalidade. O Ledger utiliza subscrições escalonadas a partir de 97 euros/mês. Preços completos na página de cada produto.
Isto é seguro para os meus clientes? +
Sim. Os clientes pagam a partir da tua própria carteira para um endereço de checkout que geramos por encomenda. Nunca fornecem à SouthPay a tua frase inicial, o teu cartão ou qualquer credencial confidencial. A transação é on-chain e verificável por eles.
Qual é o risco de volatilidade? +
Zero se liquidar em USD. O preço que o teu cliente vê no checkout fica bloqueado por um curto período e nós convertemos de acordo com a taxa que lhe mostramos. Se decidir manter a encriptação, assumirá quaisquer movimentos de preços que aconteçam depois disso.
Preciso de entender crypto? +
Não. Não é necessário qualquer conhecimento de crypto. Aceita o pagamento e nós enviamo-lo para a tua conta bancária de acordo com o plano que escolheres. A complexidade está do nosso lado.
Preciso de ter várias carteiras? +
Não. Fazemos um único pagamento para qualquer carteira (ou conta bancária) que já esteja a utilizar - mesmo que os teus clientes paguem com mais de 60 criptomoedas diferentes em diversas cadeias.
A minha categoria é "alto risco". Vai me embarcar? +
Provavelmente sim. CBD, jogos, suplementos, mercados e comércio são apenas algumas das muitas categorias com que trabalhamos. Temos uma equipa preparada exatamente para isso - analisaremos o teu negócio e saberá de imediato se podemos trabalhar em conjunto. Book a call se tiver um caso de utilização exclusivo.
Posso utilizar os dois produtos numa conta? +
Sim. O mesmo login app.southpay.io dá acesso ao Commerce e Invoice. Ligue o que precisa, deixe o resto desligado.
Com que rapidez posso começar a receber pagamentos? +
Cinco minutos se estiver no Shopify ou no WooCommerce: instale o plugin, cole a tua carteira ou informações bancárias, está no ar. Para um checkout personalizado, a integração da API geralmente demora meio dia para o programador.
